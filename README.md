# College Event Management System

A responsive college event website built with HTML, CSS, and Bootstrap. The site helps students view upcoming events, explore the gallery, register for events, and contact event coordinators.

## Project Overview

This project includes:
- Home page with hero section, event highlights, and Bootstrap carousel
- Events page featuring six event cards with images, dates, venues, and registration buttons
- Gallery page with responsive image cards, carousel, and modal preview
- Registration page with a student registration form and success alert
- FAQ page using Bootstrap accordion for easy answers
- Contact page with a contact form, coordinator details table, and social media buttons
